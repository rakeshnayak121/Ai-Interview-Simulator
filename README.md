# AI Interview Simulator

AI Interview Simulator is a full-stack web application that helps users prepare for technical and HR interviews using AI-generated questions and automated performance evaluation.

## Live Demo

https://ai-interview-simulator-4pel.vercel.app/

## Features

* Resume Upload and Analysis
* AI-Generated Interview Questions
* Technical and HR Interview Simulation
* Voice-Based Interview Support
* AI Follow-Up Questions
* Performance Evaluation and Feedback
* Interview History Tracking
* User Authentication and Authorization
* Interactive Dashboard and Analytics

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router

### Backend

* Node.js
* Express.js

### AI Integration

* Google Gemini AI
* Resume Parsing

## Installation

### Clone Repository

git clone https://github.com/rakeshnayak121/Ai-Interview-Simulator.git

### Backend Setup

cd backend

npm install

npm start

### Frontend Setup

cd frontend

npm install

npm run dev
